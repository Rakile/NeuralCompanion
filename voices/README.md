# Voice Reference Files

Place user-provided `.wav` reference voices here if you want local TTS
backends such as Chatterbox to clone a specific speaking style.

The public repository intentionally does not ship voice samples. Voice files
can contain biometric/personality likeness data, and contributors should only
add files they have the right to use and redistribute.

Recommended format:

- WAV
- short, clean speech sample
- no background music
- no copyrighted or celebrity/personality imitation unless you have rights
